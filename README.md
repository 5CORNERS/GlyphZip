# GlyphZip

Нативное Android-приложение для оптимизации кириллического текста путем замены букв на визуально схожие символы из латиницы. Основная цель — сокращение размера сообщения в байтах для сетей с жесткими ограничениями (например, Meshtastic, MeshCore, SMS), где символы кириллицы занимают вдвое больше места, чем латиница.

## Основные возможности

*   **Оптимизация текста:** Введите или вставьте свой текст и получите его оптимизированную версию с подсчетом байтов до и после.
*   **Гибкие настройки:** Приложение предлагает несколько уровней замены символов, которые можно включать и отключать группами или по отдельности:
    *   **Омоглифы:** Безопасные замены на визуально идентичные латинские буквы (например, `а`, `о`, `с`).
    *   **Курсивный стиль:** Замены, имитирующие курсивное написание (`т -> m`, `д -> g`).
    *   **Агрессивный режим:** Более радикальные замены для максимального сжатия (`ч -> 4`, `ш -> w`).
*   **Умные подсказки:** Если введенный текст превышает установленный лимит, приложение само предложит включить наиболее эффективные группы замен для его сокращения.

## Логика конфликтующих правил

В приложении существуют два правила для замены буквы «**т**»:
1.  `т -> m` (в группе "À la курсив")
2.  `т -> t` (в группе "Агрессивно")

Для корректной работы эти правила имеют строгий приоритет:

*   Правило **`т -> m`** является доминирующим. При включении этого правила (или всей его группы), правило `т -> t`, буде включено, автоматически отключается.
*   Правило **`т -> t`** можно включить только явным образом, то есть тапнув непосредственно на его чекбокс. Это действие, в свою очередь, принудительно отключит правило `т -> m`, если оно было включено. При включении всей группы "Агрессивно" общим чекбоксом, правило `т -> t` не активируется, если уже активно правило `т -> m`.
